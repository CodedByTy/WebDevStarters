# WebDev Starters

## This extension pack provides access to the best available extensions for web development.

## This extension pack includes: 
- Live Server (Microsoft)
- Tabnine (Tabnine)
- Auto Rename Tag (Jun Han)
- Prettier (Prettier)
- Better Comments (Aaron Bond)
